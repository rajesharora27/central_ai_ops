# Global Implementation Checklist

1. At startup and on context refresh, read `/docs/CONTEXT.md`, `/docs/CONTRIBUTING.md`, and `/docs/APPLICATION_BLUEPRINT.md` when present.
2. Confirm scope and impacted files.
3. Apply minimal edits in the correct layer.
4. Run targeted checks/tests.
5. Verify no stale references or backup artifacts remain.
6. Summarize changes, validation results, and any residual risks.
